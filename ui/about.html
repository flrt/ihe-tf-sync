ihe-sync is a simple tool to keep locally all the documents in sync. Please, visit the IHE website.

It's an open source project, by Frédéric Laurent, hosted on Github. Feel free to contact me, report bug or ask for improvements.

These software is distributed under GPL License.

It uses :

  • Qt
  • pyqt5
  • requests
  • beautifulsoup4
  • html5lib
